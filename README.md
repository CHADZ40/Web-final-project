## About

**Explore Cambodia** is a five-page tourism website built as a final project for **Web Development I**. The site introduces visitors to Cambodia's destinations, food, culture, and travel tips by covering iconic places like [Angkor Wat](https://en.wikipedia.org/wiki/Angkor_Wat), [Oudong mountain](https://en.wikipedia.org/wiki/Oudong), [Phnom Penh](https://en.wikipedia.org/wiki/Phnom_Penh), and [Koh Rong](https://en.wikipedia.org/wiki/Koh_Rong).

The project demonstrates semantic **HTML5** structure, external **CSS** styling, Flexbox/Grid layouts, responsive design, and interactive elements like forms and multimedia content.

## Project Plan

### Pages

- [ ] Home — Introduction to Cambodia with highlights of popular attractions
- [ ] Destinations — Famous places (Angkor Wat, Oudong Mountain, Phnom Penh, Koh Rong)
- [ ] Food & Culture — Traditional Cambodian foods and cultural activities
- [ ] Travel Tips — Transportation, costs, and recommended travel seasons
- [ ] Contact — Feedback/question form

### CSS Requirements

- [ ] External CSS stylesheet linked to all pages
- [ ] At least one `@import` (e.g. Google Fonts)
- [ ] Consistent typography across all pages
- [ ] Colors, backgrounds, and styled borders/buttons
- [ ] CSS box model (margin, padding, border)
- [ ] Flexbox and/or CSS Grid for layout
- [ ] Link states (hover, visited, active)
- [ ] Styled table and form elements

### Responsive Design

- [ ] At least two media queries (mobile, tablet, desktop)
- [ ] Responsive navigation, images, and layout
- [ ] Relative units (`%`, `em`, `rem`, `vw/vh`)

### Extra Credit (Optional)

- [ ] CSS animations or transitions
- [ ] Creative CSS Grid layouts
